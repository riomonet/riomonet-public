Natural and Mechanical Projection

#+beginverse In natural Projections onto our retinas and in cameras happens according to

  1. Laws of Reflection( which we will go into later) The law of reflection describes the behavior of light when it reflects off a surface. According to this law: The angle of incidence (the angle between the incoming light ray and the normal, or perpendicular line, to the surface) is equal to the angle of reflection (the angle between the outgoing light ray and the normal). Both the incident ray (incoming light ray) and the reflected ray (outgoing light ray) lie in the same plane as the surface.
  2. Our or the camera's position in relationship to the reflected rays. (camera coming soon)

In mechanical projection, ie drawing, painting, computer graphics, There are 3 primary methods for executing a mechanical projection

  • orthographic https://www.youtube.com/watch?v=SdLegfoMXNA Orthographic means perpendicular. An orthographic Projection is one in which all the rays of light are reflected from their surfaces and travel in parallel to each other and all intersect the picture plane at 90 degree angles.

    These are especially useful for technical and engineering drawings, because the picture is an accurate representation of the size and shape of the object. There is no distortion of shapes as in natural projections.

In computer graphcis we need to create mathamaticals models of 3d objects

  • 3d object can be thought of as:
    • 3d solids or volumes
    • 3d space

The mathamatical containers we are going to use to model and transform our 3 dimensional world are called vectors and scalars.

A scalar is a single value that represents a single attribute of an object. eg. the weight of an object the length of a an edge the time somthing took the speed at which something is traveling

A vector is an ordered list of values that represent more than a single attribute of the object it is abstracting

A vector can be thought of as an arrow pointing in a direction from an origin with a certain maginitude(distance, velocity, force). So if we have a 3 dimensional vector, the information it stores and conveys is: the direction it is pointing and it's length(magnitude).

Abstract Data Types

We are going to use the typedef operator in C to define our own abstract data types that will be used to store vector objects. Our dataypte is going to be a struct.

How to use typedef

Is a facility for creating new datatype names. for example

typedef int Length;

makes the name Length a synonym for int. Then the type Length can be used in declarations, casts, etc.. exactly the same way that the type int can.

Length len, maxlen; Length *lengths[];
