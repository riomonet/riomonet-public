Vector Transformations

Right now we have a cube with static points. they Don't change, they don't move.
To give movement to a 3d object we apply transformations to the models.
These are known as linear tranformations.
linearinput(vector) => linearoutput(vector)
vectors are lines pointin in a direction with a magnitude.

They are 3 main transformations we can apply to a vector

scaling -> change the magnitude multiply each vector by scalar
translations -> move to a different position add to each vector by a scalar
rotation -> point towards a different direction.
                rotate on a given axis. around x,y,z

We already are performing scaling and translation

- scaling => we apply by multiplying each x and y of a vec2t by a
scalar. remember our model is built on values from -1 to 1.
So we apply a scaling transformation to every vector in our model.
- translation we apply already by adding a scalar to every vector.

rotation matrix formula derivation

we will start by rotating vectos in 2 dimensions


If we draw a vector p(x,y) with lenght r
we have we have cos(alpha) = x/r
and we have the sin(alpha) = y/r

rememember r, the length of the vector,
will remain constant in a rotation.

if we do a little aglgebr we can come up with
x = r*cos(alpha)
y = r*sin(alpha)

we want to find p'(x',y') with an angle of
alpha + beta.

r is still the length of p' because remember
in rotations the length remains constant.

for angle(alpha + beta) adjacent is x' and opposite is y'
so cos(alpha + Beta) = x'/r
and sin(alpha +beta) = y'/r
applying the same algebra as aobve.
x' = r*cos(alpha + beta)
y' = r*sin(alpha + beta)

  • Finding x' with the Angle addition formula for cosine

    we find x' using angle addition formual for cosine

    • cos(a + b) = cosA*cosB - sinA*sinB

      so we have

      cos(alpha + Beta) = x'/r =>
      x' = r*cos(alpha + Beta) =>

      using angle addition formual for cosine =>

      x' = r * (cos(alpha)*cos(Beta) - sin(alpha)*sin(Beta)) =>
      x' = r * cos(alpha)*cos(Beta) - r * sin(alpha)*sin(beta)

      well we know from previously that
      r * cos(alpha) = x
      r * sin(alpha) = y

      so we have.

      x' = x*cos(beta) - y * sin(Beta)

      This is the formula we will be using for our rotation matrix

  • Finding y' with the Angle addition formula for sin
    • sin(a + b) = sin(a) * cos(b) + cos(a) * sin(b)

      y' = r*sin(alpha + beta) =>

      y' = r * (sin(alpha)*cos(b) + cos(alpha)*sin(beta)) =>

      y' = r *sin(alpha)*cos(b) + r * cos(alpha)*sin(beta) =>

      and we know from previously that

      x = r*cos(alpha)
      y = r*sin(alpha)

      so substituting into the formula we have

      y' = y*cos(beta) + x*sin(beta)

rotation matrix formula

x' = x*cos(beta) - y * sin(Beta)
y' = y*cos(beta) + x * *sin(beta)

  • Rotation functions
    vec3_t vec3_rotate_z(vec3_t v, float angle)
{
  vec3_t rotated_vector = {
    .x = v.x * cos(angle) - v.y * sin(angle),
    .y = v.x * sin(angle) + v.y * cos(angle),
    .z = v.z
  };
  return rotated_vector;
}

  vec3_t vec3_rotate_y(vec3_t v, float angle)
{
  vec3_t rotated_vector = {
    .x = v.x * cos(angle) - v.z * sin(angle),
    .y = v.y
    .z = v.x * sin(angle) + v.z * cos(angle),
  };
  return rotated_vector;
}

vec3_t vec3_rotate_x(vec3_t v, float angle)
{
  vec3_t rotated_vector = {
    .x = v.x;
    .y = v.y * cos(angle) - v.z * sin(angle),
    .z = v.y * sin(angle) + v.z * cos(angle),
  };
  return rotated_vector;
}
